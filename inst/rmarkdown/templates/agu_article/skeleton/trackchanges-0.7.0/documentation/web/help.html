TrackChanges Help
Overview



The TrackChanges package adds the ability to collaboratively edit LaTeX documents. It allows multiple editors to make changes and add notes to a document. These changes and notes will be color coded and added to the compiled document. TrackChanges also includes a tool that an author can use to quickly find and take action on the suggested edits.

The TrackChanges package consists of two parts:
  1. A LaTeX style file, trackchanges.sty.
  2. A GUI used to accept/reject/change edits.
